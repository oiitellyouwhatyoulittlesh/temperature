# TEMPERATURE CONVERTER THINGY

## features or whatever
* Changes Fahrenheit into Celsius if you live in America.
* Changes Celsius into Fahrenheit if you live anywhere normal.
* Yells at you in bright red text if you type regular letters instead of numbers.
* Refuses to work if you go past absolute zero because science says that is illegal.
* Stacks three whole frames on top of each other like a deck of cards.
* Has a reset button to instantly clear your mistakes.
* Stretches and grows when you resize the window so it does not look completely broken.

## how to run it
1. Make sure Python is actually installed on your computer.
2. Open the file and run it.
3. Type some random numbers and click calculate.
